# Almoxarifado — Controle de Estoque

Sistema web para controle de estoque de insumos de saúde, desenvolvido como projeto da disciplina de ADS.

## 🔗 Link do Projeto

> [Acesse o projeto aqui](https://universidade-cesumar.github.io/prova-2bi-ads-3sem-vinichaveer-beep/)

## 📋 Funcionalidades

- Cadastro de materiais com nome e quantidade (POST)
- Listagem de todos os materiais em estoque (GET)
- Filtro de busca em tempo real por nome
- Contador de total de itens no dashboard
- Alerta visual para itens com estoque crítico (menos de 10 unidades)
- Tratamento de erros de conexão com a API

## 🛠️ Tecnologias

- HTML5, CSS3, JavaScript (Vanilla)
- MockAPI (https://mockapi.io) — API REST para persistência dos dados
- GitHub Pages — hospedagem

## 📁 Estrutura

```
index.html   → estrutura da página
style.css    → estilos
main.js      → lógica de comunicação com a API
```

## 🚀 Como executar localmente

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador
